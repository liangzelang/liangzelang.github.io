leetcode 1230.抛掷硬币

解题思路

  1. 这种类型的题目显然可以优化成最小结构，使用动态规划
  2. 最小结构： dp[i][j]: 抛掷i次硬币，j次正面的概率
  3. 状态转移方程： dp[i][j] = dp[i-1][j]*(1-(prob[i-1])) + dp[i-1][j-1]*prob[i-1] （注意这里第i次的概率对应数组index要-1）
  4. 边界条件分析，dp[0][0]=1

复杂度分析

  • 时间复杂度：O(n^2)
  • 空间复杂度：O(n^2) 认为row和col都为n
